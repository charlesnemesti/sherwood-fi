# Sherwood Fi

Onchain real-world asset marketplace for tokenized tax liens and tax deeds on Robinhood Chain.

## Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Optimized for [Vercel](https://vercel.com). Push to GitHub and import the repository in Vercel.
